================================================================================
CLOUD9 ERP PHASE 7 - START LOCAL SERVER
================================================================================

OPTION 1: SIMPLEST WAY (Recommended for Testing)
================================================================================

Just run this ONE command in PowerShell:

    python local_server.py

That's it! The server will start on:
    
    🌐 http://localhost:8000/

Then you can:
    1. Open browser to http://localhost:8000/docs
    2. See all Phase 7 endpoints with interactive testing
    3. Test health checks, reports, caching, optimization

================================================================================
WHAT YOU'LL SEE
================================================================================

When you run the command, you'll see:

    ================================================================================
    Cloud9 ERP Phase 7 - Local Demo Server
    ================================================================================
    ✅ Starting local demo server (no database required)
    
    Phase 7 Features Available:
      ✅ PDF Report Generation
      ✅ Excel Report Generation
      ✅ API Caching Layer
      ✅ Scheduled Report Runner
      ✅ Database Optimization
      ✅ System Health Monitoring
    
    Access the API:
      🌐 API Docs (Swagger):   http://localhost:8000/docs
      📖 ReDoc Docs:           http://localhost:8000/redoc
      ❤️  Health Check:        http://localhost:8000/health/
      📊 System Metrics:       http://localhost:8000/health/metrics
    
    Press Ctrl+C to stop the server

================================================================================
TEST IT IN BROWSER
================================================================================

1. Open: http://localhost:8000/docs

2. You'll see all endpoints:
   - Health checks
   - Report generation
   - Cache statistics
   - Optimization status

3. Click "Try it out" on any endpoint to test

4. See responses in real-time

================================================================================
TEST IT IN PowerShell
================================================================================

While server is running, open NEW PowerShell window and run:

    # Test health
    curl http://localhost:8000/health/

    # Test metrics
    curl http://localhost:8000/health/metrics

    # Test reports
    curl http://localhost:8000/api/v1/reports/demo

    # Test cache
    curl http://localhost:8000/api/v1/cache/stats

    # Test optimization
    curl http://localhost:8000/api/v1/optimization/health

================================================================================
STOP THE SERVER
================================================================================

In the PowerShell window where server is running:

    Press Ctrl + C

The server will shut down gracefully.
